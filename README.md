# Hindi POS Tagging (NLP)

This project demonstrates Part-of-Speech (POS) tagging for Hindi language text using Stanza (Stanford NLP).

## Features
- Hindi text tokenization
- Hindi POS tagging using trained models
- Universal POS tagset

## Tools Used
- Python
- Stanza (Stanford NLP)
- Google Colab

## Description
Stanza provides pretrained Hindi language models that enable accurate POS tagging.
Universal POS tags are used to maintain consistency across languages.

## Purpose
This project is created to understand and apply Hindi NLP techniques using industry-standard tools.
